One more step — deploy your web app

Your sheets are ready. Follow these steps to launch the cardBalance web app:

1
Open the Script Editor
Click Extensions → Apps Script in the menu bar above.
2
Create a New Deployment
Click Deploy → New deployment. Click the gear icon and select Web app.
3
Configure Access
Set Execute as: Me and Who has access: Only myself (or Anyone if sharing). Click Deploy.
4
Authorize & Copy Your URL
Click Authorize access and follow the prompts. Copy the web app URL — bookmark it for easy access.
💡 Your data is stored entirely in this spreadsheet. The web app URL is your personal access link — keep it safe.

How to update

1
Open the Script Editor
Click Extensions → Apps Script.
2
Replace Code.gs
Open Code.gs on GitHub, copy all contents, and paste into your Code.gs file. Save with Ctrl+S.
3
Replace Index.html
Open Index.html on GitHub, copy all contents, and paste into your Index.html file. Save.
4
Redeploy
Click Deploy → Manage deployments. Click the pencil icon, set version to New version, and click Deploy.
💡 Your data is never affected by updates — it lives in the spreadsheet, not the script.
